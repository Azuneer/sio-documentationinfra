# Template — Documentation infrastructure réseau et systèmes

Template de documentation d'infrastructure réseau et systèmes, conçue pour la section SIO du lycée Paul-Louis Courier (Tours). Destinée à être complétée par les enseignants.
---

## Contenu du dépôt

| Fichier | Description |
|---------|-------------|
| `documentation_infrastructure.md` | Version complète — topologie, fiches équipements, services, procédures, glossaire |
| `documentation_infrastructure_light.md` | Version synthétique — VLANs, adressage, configurations commutateurs, accès |

---

## Utilisation

1. Choisir la version adaptée au besoin (complète ou synthétique).
2. Rechercher les champs `_À compléter_` et les renseigner.
3. **Ne jamais écrire de mot de passe en clair** — référencer uniquement le gestionnaire de secrets de la section.
4. Mettre à jour le tableau "Historique des modifications" à chaque révision.

---

## Avertissement

Aucun identifiant, mot de passe ou secret ne doit être commité dans ce dépôt.
